// To create the actual widget in Unreal Engine 5:
// 1. Right-click in Content Browser -> User Interface -> Widget Blueprint
// 2. Name it "WBP_VirtualJoystick"
// 3. Add a Canvas Panel as root
// 4. Add two Image widgets: one for joystick background, one for joystick knob
// 5. Add the following logic:

/*
Event Graph Logic:

Variables:
- JoystickCenter (Vector2D)
- MaxRadius (float) = 100.0
- AxisValue (Vector2D)
- bIsPressed (bool)

Functions:
- SetupJoystick()
  - Get viewport size
  - Set joystick position to bottom-left or bottom-right corner
  - For mobile: position based on first touch
  - For desktop: fixed position

- OnTouchStarted (TouchPosition: Vector2D)
  - Set JoystickCenter = TouchPosition
  - Set bIsPressed = true
  - Call VirtualJoystickComponent->OnTouchStarted(TouchPosition)

- OnTouchMoved (TouchPosition: Vector2D)
  - Calculate Delta = TouchPosition - JoystickCenter
  - Clamp Delta length to MaxRadius
  - Update knob position = JoystickCenter + Delta
  - Normalize AxisValue = Delta / MaxRadius
  - Call VirtualJoystickComponent->OnTouchMoved(TouchPosition)

- OnTouchEnded()
  - Reset knob position to JoystickCenter
  - Set AxisValue = (0, 0)
  - Set bIsPressed = false
  - Call VirtualJoystickComponent->OnTouchEnded()

Tick Event:
- If bIsPressed and using keyboard fallback:
  - Read WASD/Arrow keys or gamepad stick
  - Update knob visual position based on input
  - Update AxisValue accordingly
*/
