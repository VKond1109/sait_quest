# UE5 Ultimate Game - Полная документация

## Обзор проекта

Это полноценный фреймворк для создания AAA-игры на Unreal Engine 5 с поддержкой ПК и мобильных платформ. Проект включает все современные механики: от виртуального джойстика до сложной системы репутации и строительства.

## Реализованные системы

### 1. Базовые системы (Core)
- **Виртуальный джойстик** - поддержка touch, клавиатуры и геймпада
- **Персонаж** - ходьба, бег, прыжки, двойной прыжок, рывок (dash)
- **Здоровье** - урон, лечение, регенерация, смерть
- **Выживание** - голод, жажда, выносливость, температура

### 2. Боевые системы (Combat)
- **HealthComponent** - управление здоровьем и уроном
- **Боевая система** - атака, блокирование, уклонение
- **Враги с ИИ** - патрулирование, преследование, атака

### 3. RPG системы (Gameplay)
- **Инвентарь** - сетка предметов, перетаскивание, стаки
- **Крафт** - создание предметов из ресурсов
- **Квесты** - цепочки заданий, награды, отслеживание
- **Диалоги** - ветвящиеся разговоры с NPC
- **Дерево навыков** - прокачка характеристик и умений
- **Кастомизация** - изменение внешности персонажа

### 4. Экосистема (Ecosystem) ⭐ НОВОЕ
- **Животные и существа** - травоядные, хищники, всеядные
- **Пищевая цепочка** - охота, бегство, поиск пищи
- **Поведение ИИ** - блуждание, сон, реакция на угрозы
- **Динамическая популяция** - рождение, смерть, миграция

### 5. Экипировка (Equipment) ⭐ НОВОЕ
- **Система брони** - 8 слотов экипировки
- **Типы брони** - легкая, средняя, тяжелая, щит
- **Прочность** - износ и ремонт предметов
- **Визуальные изменения** - смена мешей при экипировке
- **Бонусы к характеристикам** - сила, ловкость, сопротивление

### 6. Транспорт (Vehicles) ⭐ НОВОЕ
- **Базовый класс транспорта** - автомобили, мотоциклы, танки
- **Физика движения** - ускорение, торможение, повороты
- **Топливо** - расход, заправка, остановка без топлива
- **Пассажиры** - водитель и пассажиры
- **Боевой транспорт** - стрельба из танков
- **Ездовые животные** - альтернатива механизмам

### 7. Строительство (Building) ⭐ НОВОЕ
- **Режим строительства** - включение/выключение
- **Типы элементов** - стены, полы, потолки, крыши, двери, окна
- **Материалы** - дерево, камень, металл, бетон, стекло
- **Сетка привязки** - точное размещение
- **Ресурсы** - проверка и потребление материалов
- **Ремонт и снос** - восстановление и демонтаж

### 8. Репутация и фракции (Reputation) ⭐ НОВОЕ
- **Отношения с фракциями** - 6 уровней от союзника до врага
- **Репутация** - очки от -1000 до 1000
- **Известность (Notoriety)** - влияние на реакцию NPC
- **Слухи** - распространение информации
- **Преступления и добрые дела** - влияние на репутацию
- **Наемники** - найм помощников
- **Модификаторы цен** - скидки и наценки в зависимости от отношений
- **Уникальные преимущества (Perks)** - разблокировка способностей

### 9. Окружение (Environment)
- **Погода** - дождь, снег, туман, ясная погода
- **Время суток** - динамическая смена дня и ночи
- **Разрушаемость** - Chaos Destruction для объектов

### 10. Сеть (Network)
- **Мультиплеер** - кооператив и соревновательный режимы
- **Синхронизация** - позиций, действий, состояния мира

### 11. Интерфейс (UI)
- **Главное меню** - начало игры, настройки, выход
- **HUD** - здоровье, выносливость, мини-карта, цели
- **Инвентарь** - управление предметами
- **Пауза** - настройки, сохранения, выход в меню

### 12. Аудио (Audio)
- **Динамическая музыка** - смена в зависимости от ситуации
- **Пространственный звук** - 3D аудио
- **Реверберация** - эффекты окружения

### 13. Доступность (Accessibility)
- **Режим для дальтоников** - коррекция цветов
- **Субтитры** - настраиваемый размер
- **Локализация** - поддержка множества языков

## Установка и сборка

### Требования
- Unreal Engine 5.3 или новее
- Visual Studio 2019/2022 (для Windows)
- Xcode (для macOS/iOS)
- Android NDK (для Android)

### Шаги по установке

1. **Клонирование проекта**
   ```bash
   cd /workspace/UE5UltimateGame
   ```

2. **Генерация проектов**
   - Windows: Кликните правой кнопкой на `.uproject` → "Generate Visual Studio project files"
   - Или через командную строку:
     ```bash
     "C:\Program Files\Epic Games\UE_5.3\Engine\Build\BatchFiles\GenerateProjectFiles.bat"
     ```

3. **Компиляция C++**
   - Откройте решение в Visual Studio
   - Выберите конфигуляцию "Development Editor"
   - Нажмите F7 для сборки

4. **Запуск в редакторе**
   - Дважды кликните на `UE5UltimateGame.uproject`
   - Дождитесь компиляции шейдеров

### Сборка для платформ

#### Windows
```bash
"C:\Program Files\Epic Games\UE_5.3\Engine\Build\BatchFiles\RunUAT.bat" BuildCookRun \
  -project="UE5UltimateGame.uproject" \
  -platform=Win64 \
  -clientconfig=Shipping \
  -serverconfig=Shipping \
  -build
```

#### Android
1. Настройте Android SDK и NDK в настройках редактора
2. Включите поддержку Android в проекте
3. Соберите через File → Package Project → Android

#### iOS
1. Установите Xcode и инструменты разработчика Apple
2. Настройте сертификаты подписи
3. Соберите через File → Package Project → iOS

## Использование систем в Blueprint

### Пример: Добавление экосистемы существу

1. Создайте Blueprint класса `Actor`
2. Добавьте компонент `EcosystemComponent`
3. Настройте параметры:
   - Creature Type: Herbivore/Carnivore/Predator
   - Max Speed: 500
   - Health: 100
4. В Event BeginPlay инициализируйте поведение

### Пример: Система строительства

```blueprint
Event B -> Call EnterBuildMode()
Event Mouse Scroll -> Call SelectElement(Wall, Wood)
Event Left Click -> Call TryBuildElement(HitLocation, HitRotation)
Event R -> Call RotateElement()
Event Right Click -> Call ExitBuildMode()
```

### Пример: Изменение репутации

```blueprint
// После убийства NPC дружественной фракции
Call CommitCrime("Faction_Guards", Severity=5)

// После помощи NPC
Call PerformGoodDeed("Faction_Merchants", Merit=3)

// Проверка отношения
If GetRelationWithFaction("Faction_Guards") >= Friendly
    -> Разблокировать диалоги
Else
    -> NPC атакуют
```

## Управление

### ПК
- **WASD** - Перемещение
- **Shift** - Бег
- **Пробел** - Прыжок (x2 для двойного прыжка)
- **Ctrl** - Рывок (Dash)
- **ЛКМ** - Атака / Взаимодействие
- **ПКМ** - Блокировка / Прицеливание
- **E** - Взаимодействие
- **I** - Инвентарь
- **K** - Дерево навыков
- **Q** - Режим строительства
- **R** - Поворот элемента строительства
- **T** - Переключение камеры (1-е/3-е лицо)
- **~** - Консоль разработчика

### Мобильные
- **Левый стик** - Перемещение
- **Правый стик** - Камера
- **Кнопка A** - Прыжок
- **Кнопка B** - Рывок
- **Кнопка X** - Атака
- **Кнопка Y** - Взаимодействие
- **Свайп двумя пальцами** - Камера

### Геймпад
- **Левый стик** - Перемещение
- **Правый стик** - Камера
- **A/Cross** - Прыжок
- **B/Circle** - Рывок
- **X/Square** - Взаимодействие
- **Y/Triangle** - Атака
- **LB/L1** - Блокировка
- **RB/R1** - Прицеливание
- **Start** - Пауза

## Производительность и оптимизация

### Для мобильных устройств
- Используйте LOD для всех моделей
- Оптимизируйте текстуры (макс. 2048x2048)
- Отключите пост-процесс эффекты
- Уменьшите количество динамических теней
- Используйте Instanced Static Meshes для повторяющихся объектов

### Для ПК
- Включите Nanite для детализированных моделей
- Используйте Lumen для глобального освещения
- Настройте уровни детализации под целевое железо

## Расширение функционала

### Добавление нового типа транспорта
1. Создайте класс, наследующий от `AVehicleBase`
2. Переопределите `SetupVehicle()` для настройки физики
3. Добавьте уникальные способности (например, полет для ховеркрафта)

### Создание новой фракции
1. В Blueprint создайте Data Asset с информацией о фракции
2. Настройте начальные отношения и уникальные perks
3. Добавьте NPC этой фракции на уровень

### Новый элемент строительства
1. Создайте префаб здания
2. Добавьте запись в таблицу данных `DT_BuildingElements`
3. Настройте стоимость и требования к ресурсам
